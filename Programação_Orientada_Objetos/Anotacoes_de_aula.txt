Atributo - Caracteristicas de um objeto, essas caracteristicas também são conhecidas como variáveis, utilizando o exemplo de pessoa, temos alguns atributos, tais como:
nome, idade, endereço, peso, altura, etc.
public class Pessoa{
//Atributos
public String nome;
public int idade;
public String endereco;
public float peso;
public float altura;
}



Classe - É um tipo definido pelo usuário que contém a "receita", a especificação para os objetos, a classe envolve, associa, funções e dados, controlando o acesso a estes, defini-la implica em especificar os seus atributoss
(dados) e seus métodos(funções). Possuem nome, visibilidade(public, private), caracteristicas e ações. Para criar uma classe basta declarar a visibilidade, digitar a palavra reservada classe, abrir e fechar chaves.
public class  Teste{
// Atributos ou Propriedades 

//Métodos
}
Instância - é a declaração de um objeto que foi definido por uma classe.

Objeto - São caracterizados por atributos e métodos. Atributos são as propriedades de um objeto, Métodos são as ações que um objet pode realizar. Os objetos são características definidas pelas classes. Neles é permitido instanciar objetos da classe

Encapsulamento - Encapsulamento vem de encapsular, que em programção orientada a objetos, significa separa o programa em partes, o mais isolado possivel A idéia é tornar o software ,aos flexível, fácil de modificar e de criar novas implementações. O encapsulamento 
serve para controlar o acesso aos atributos e métodos de uma classe.

Método Construtor - Os contrutores são responsáveis por criar o objeto em memória, ou seja, instanciar a classe que foi definida. Eles são obrigatórios e são declarados conforme o exemplo abaixo:
public class Carro{

public Carro(){

}
}

Para criar um objeto da classe Carro simplismente usamos a palavra reservada new e o nosso construtor é chamado. Também é possivel utilizar construtores da classe mãe com a palavra reservada super();
